Поиск кодов ошибок восстановления iTunes / iOS

Видите ошибку при восстановлении или обновлении iPhone через iTunes или aisi-helper? Введите код (напр. 4013, 9, 3194, 4014) — сначала решите по официальной сортировке Apple; если все программные шаги не помогли, посмотрите причину уровня ремонта, чтобы понять, аппаратная ли это неисправность.

Шаг 1: решение по официальной сортировке (программный уровень)

Apple группирует ошибки обновления/восстановления по направлению сортировки; выполняйте их по порядку, и большинство ошибок подключения/сети устраняются сами. Для настольного средства с ремонтом драйверов в один клик и встроенным помощником по кодам ошибок попробуйте средство кодов ошибок iTunes от aisi-helper и руководство по прошивке / восстановлению «кирпича» в один клик; если устройство вовсе не подключается к компьютеру, см. iPhone не подключается к компьютеру.

Проверка ПО для обеспечения безопасности2 / 4 / 6 / 9 / 1611 / 9006
Отключите или настройте сторонний антивирус и брандмауэр, чтобы они не мешали компьютеру обращаться к серверам Apple, и повторите обновление.
Проверка подключений через USB13 / 14 / 1600-1604 / 2000-2009 / 4005 / 4013 / 4014
Используйте оригинальный или MFi-кабель и задний порт USB (не на клавиатуре и не через хаб); если не помогает, смените кабель, порт и компьютер.
Проверка сети / серверов Apple17 / 1638 / 3000-3020 / 3194 / 3200
Смените сеть, проверьте файл hosts и отключите прокси; 3194 обычно означает понижение до неподписанной прошивки — восстановите последнюю подписанную сборку.
Проверка аппаратного обеспечения1 / 3 / 6 / 11 / 12 / 16 / 20-56 / 1667 / 1669
С исправными кабелем, компьютером и сетью восстановите ещё дважды; если повторяется, обычно это аппаратная неисправность, требующая профессионального ремонта.
Джейлбрейк / понижение / устаревшее ПО1015, «Не удаётся найти требуемый ресурс.»
Возникает на устройствах со взломом или понижением версии, а также при устаревшем iTunes — обновите и используйте режим восстановления для последней iOS.
ПО устройства слишком новое10
iOS устройства новее, чем iTunes на компьютере — обновите macOS / iTunes / приложение «Устройства Apple».

Шаг 2: всё ПО не помогло → диагностика уровня ремонта (аппаратный уровень)

Если тот же код повторяется после двух восстановлений с исправными кабелем, компьютером и сетью, обычно это оборудование. Схема ниже сопоставляет код ошибки → узел системной платы с точки зрения ремонтного стенда (только иллюстративно; реальная диагностика требует измерения сопротивления/тока). Аппаратный ремонт доверяйте специалистам.

Быстрая сортировка реальных случаев

  • Зависает около 80 % при восстановлении: 80 % — обычно этап проверки после записи; обеспечьте заряд батареи ≥50 % и стабильное напряжение, чтобы проверка не сорвалась.
  • Ошибка 9 появляется только спустя долгое время после отключения кабеля: память не считывается (холодная пайка/питание), поэтому повтор программно не помогает — готовьтесь к ремонту.
  • Ошибка 3194 / 4013 при понижении старой iOS: старая прошивка больше не подписывается, или стороннее средство переписало hosts для перенаправления gs.apple.com — восстановите hosts и прошейте последнюю подписанную прошивку.
  • Восстановленный телефон в цикле загрузки с 4013, перезапуск ровно когда ток достигает ~550 мА: линия питания buck/зарядки не работает (реальная диагностика на стенде) — оборудование.
  • Драйвер / iTunes не устанавливается со связанными ошибками: переустановка драйвера Apple Mobile Device / AMDS через ремонтный набор — самое надёжное решение.

Направления сортировки собраны из официальной документации поддержки Apple (ошибки обновления и восстановления iOS 108308 / ошибки 9·4005·4013·4014 102561); причины уровня ремонта обобщены из опыта публичных ремонтных сообществ, только для справки — при аппаратных неисправностях обращайтесь к профессиональному ремонту.
