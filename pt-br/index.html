Consulta de códigos de erro de restauração do iTunes / iOS

Vendo um erro ao restaurar ou atualizar o iPhone com iTunes ou aisi-helper? Digite o código (ex.: 4013, 9, 3194, 4014): primeiro resolva com a triagem oficial da Apple; se todas as etapas de software falharem, veja a causa no nível de reparo para saber se é falha de hardware.

Etapa 1: resolver com a triagem oficial (camada de software)

A Apple agrupa os erros de atualização/restauração por direção de triagem; siga em ordem e a maioria dos erros de conexão/rede se resolve sozinha. Para uma ferramenta de desktop com reparo de drivers em um clique e assistente de códigos de erro integrado, experimente a ferramenta de códigos de erro do iTunes do aisi-helper e o guia de flash / recuperação em um clique; se o dispositivo não conecta ao computador, veja o iPhone não conecta ao computador.

Verifique o software de segurança2 / 4 / 6 / 9 / 1611 / 9006
Desative ou ajuste o antivírus e o firewall de terceiros para que não impeçam o computador de alcançar os servidores da Apple, e tente atualizar novamente.
Verifique as conexões USB13 / 14 / 1600-1604 / 2000-2009 / 4005 / 4013 / 4014
Use um cabo original ou MFi e uma porta USB traseira (não do teclado nem hub); se ainda falhar, troque cabo, porta e computador.
Verifique a rede / servidores da Apple17 / 1638 / 3000-3020 / 3194 / 3200
Mude de rede, verifique o arquivo hosts e desative o proxy; o 3194 geralmente indica downgrade para firmware não assinado: restaure a versão assinada mais recente.
Verifique o hardware1 / 3 / 6 / 11 / 12 / 16 / 20-56 / 1667 / 1669
Com cabo, computador e rede em bom estado, restaure mais duas vezes; se persistir, geralmente é falha de hardware que precisa de reparo profissional.
Jailbreak / downgrade / software desatualizado1015, “O recurso necessário não pode ser encontrado.”
Ocorre em dispositivos com jailbreak ou em downgrade, e também com iTunes desatualizado: atualize e use o modo de recuperação para restaurar o iOS mais recente.
Software do dispositivo novo demais10
O iOS do dispositivo é mais novo que o iTunes do computador: atualize macOS / iTunes / app Dispositivos Apple.

Etapa 2: todo o software falhou → diagnóstico no nível de reparo (camada de hardware)

Se o mesmo código persiste após duas restaurações com cabo, computador e rede em bom estado, geralmente é hardware. Triagem rápida do mundo real

  • Trava por volta de 80% na restauração: 80% costuma ser a fase de verificação após a gravação; garanta bateria ≥50% e tensão estável para a verificação não falhar.
  • O erro 9 aparece só muito depois de desconectar o cabo: o armazenamento não está sendo lido (solda fria/energia), então tentar de novo por software não ajuda: prepare-se para o reparo.
  • Erro 3194 / 4013 ao fazer downgrade de um iOS antigo: o firmware antigo não é mais assinado, ou uma ferramenta de terceiros reescreveu o hosts para redirecionar gs.apple.com: restaure o hosts e instale o firmware assinado mais recente.
  • Um aparelho recondicionado em loop de inicialização com 4013, reiniciando bem quando a corrente atinge ~550 mA: o trilho de alimentação buck/carga não funciona (diagnóstico real de bancada): hardware.
  • Driver / iTunes não instala com erros relacionados: reinstalar o driver Apple Mobile Device / AMDS com um kit de reparo é a solução mais confiável.
