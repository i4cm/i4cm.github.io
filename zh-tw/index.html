iTunes / iOS 回復錯誤碼查詢

用 iTunes 或相關工具回復、更新 iPhone 時出現錯誤？輸入錯誤碼（如 4013、9、3194、4014），先依 Apple 官方排查方向自救；軟體都試過仍報，再看 維修級真因判斷是不是硬體問題。

第一步：依官方排查方向自救（軟體層）

Apple 把更新/回復錯誤依排查方向分類，照順序逐一排查，多數連線/網路類錯誤可自行解決。想要附一鍵修復驅動、內建錯誤碼小幫手的桌面工具，可用 aisi-helper iTunes 錯誤碼查詢 與 一鍵刷機 / 救磚教學；裝置完全連不上電腦，參考 iPhone 連不上電腦怎麼辦。

檢查安全防護軟體2 / 4 / 6 / 9 / 1611 / 9006
關閉或調整第三方防毒與防火牆，別讓它擋住電腦連 Apple 伺服器，然後重試更新。
檢查 USB 連線13 / 14 / 1600-1604 / 2000-2009 / 4005 / 4013 / 4014
用原廠或 MFi 連接線、機筱後方 USB 埠（別用鍵盤埠/集線器）；不行就換線、換埠、換電腦。
檢查網路 / Apple 伺服器17 / 1638 / 3000-3020 / 3194 / 3200
換網路、查 hosts、關代理；3194 多是降級到停簽韌體 → 改刷最新在簽版本。
檢查硬體1 / 3 / 6 / 11 / 12 / 16 / 20-56 / 1667 / 1669
在連接線、電腦、網路都正常時再回復兩次；仍報多為硬體故障，需送專業維修。
越獄 / 降級 / 軟體過時1015、「找不到所需資源」
越獄機或正在降級會報；iTunes 過舊也會 → 更新後用復原模式刷回最新版 iOS。
裝置軟體太新10
裝置 iOS 比電腦端 iTunes 還新，更新 Mac / iTunes / Apple 裝置 App 即可。

第二步：軟體都無效 → 維修級真因（硬體層）

若在良好的連接線、電腦、網路下連刷兩次仍報同一個碼，多半是硬體。下圖是維修視角的「錯誤碼 → 主機板部位」對應（僅示意，實際需結合電阻/電流實測）。硬體維修請交專業人員。

真實情境速判

  • 刷到 80% 左右卡住：80% 多是韌體寫入後的驗證階段，先確認電量 ≥ 50%、電壓穩定，避免驗證失敗。
  • 拔掉連接線很久後才報 9：多半沒讀到硬碟（虛焊/供電），軟體再試也無解，準備送修。
  • 降級舊版 iOS 報 3194 / 4013：舊韌體已停簽，或 hosts 被第三方工具改寫重新導向 gs.apple.com → 還原 hosts、改刷最新在簽韌體。
  • 二修機白蘋果反覆重啟報 4013、開機電流剛過 550mA 就重啟：降壓/充電電源沒正常運作（實機維修判斷），屬硬體。
  • 驅動程式 / iTunes 裝不上連帶報錯：用維修工具箱「iTunes 及驅動程式修復」一鍵重裝最穩。
