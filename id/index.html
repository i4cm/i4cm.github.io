Pencarian Kode Error Pemulihan iTunes / iOS

Melihat error saat memulihkan atau memperbarui iPhone dengan iTunes atau aisi-helper? Masukkan kode (mis. 4013, 9, 3194, 4014) - pertama perbaiki sendiri dengan triase resmi Apple; jika setiap langkah perangkat lunak gagal, periksa penyebab tingkat perbaikan untuk tahu apakah ini kerusakan perangkat keras.

Langkah 1: perbaiki sendiri dengan triase resmi (lapisan perangkat lunak)

Apple mengelompokkan error pembaruan/pemulihan menurut arah triase; kerjakan berurutan dan kebanyakan error koneksi/jaringan teratasi sendiri. Untuk alat desktop dengan perbaikan driver satu klik dan pembantu kode error bawaan, coba alat kode error iTunes aisi-helper dan panduan flash / un-brick satu klik; jika perangkat sama sekali tidak terhubung ke komputer, lihat iPhone tidak terhubung ke komputer.

Memeriksa perangkat lunak keamanan2 / 4 / 6 / 9 / 1611 / 9006
Nonaktifkan atau sesuaikan antivirus dan firewall pihak ketiga agar tidak memblokir komputer menjangkau server Apple, lalu coba perbarui lagi.
Memeriksa sambungan USB13 / 14 / 1600-1604 / 2000-2009 / 4005 / 4013 / 4014
Gunakan kabel asli atau MFi dan port USB di belakang (bukan port keyboard atau hub); jika tetap gagal, ganti kabel, port, dan komputer.
Memeriksa jaringan / server Apple17 / 1638 / 3000-3020 / 3194 / 3200
Ganti jaringan, periksa berkas hosts, dan matikan proxy; 3194 biasanya berarti downgrade ke firmware yang tidak ditandatangani — pulihkan build bertanda tangan terbaru.
Memeriksa perangkat keras1 / 3 / 6 / 11 / 12 / 16 / 20-56 / 1667 / 1669
Dengan kabel, komputer, dan jaringan yang baik, pulihkan dua kali lagi; jika tetap, biasanya kerusakan perangkat keras dan perlu perbaikan profesional.
Jailbreak / downgrade / perangkat lunak usang1015, 'Sumber daya yang diperlukan tidak dapat ditemukan.'
Terjadi pada perangkat yang di-jailbreak atau di-downgrade, juga iTunes usang — perbarui, lalu gunakan mode pemulihan untuk memulihkan iOS terbaru.
Perangkat lunak perangkat terlalu baru10
iOS perangkat lebih baru daripada iTunes komputer — perbarui macOS / iTunes / app Perangkat Apple.

Langkah 2: semua perangkat lunak gagal → diagnosis tingkat perbaikan (lapisan perangkat keras)

Jika kode sama bertahan setelah dua kali pemulihan dengan kabel, komputer, dan jaringan yang baik, biasanya perangkat keras. Diagram di bawah memetakan kode error → bagian papan logika dari sudut bangku perbaikan (ilustrasi saja; diagnosis nyata perlu pengukuran resistansi/arus). Serahkan perbaikan perangkat keras ke profesional.

Triase cepat dunia nyata

  • Macet di sekitar 80% saat pemulihan: 80% biasanya tahap verifikasi setelah penulisan — pastikan baterai ≥50% dan tegangan stabil agar verifikasi tidak gagal.
  • Error 9 baru muncul lama setelah kabel dicabut: penyimpanan tidak terbaca (solder dingin/daya), jadi mengulang lewat perangkat lunak tidak membantu — siapkan perbaikan.
  • Error 3194 / 4013 saat downgrade iOS lama: firmware lama tidak lagi ditandatangani, atau alat pihak ketiga menulis ulang hosts untuk mengalihkan gs.apple.com — pulihkan hosts dan flash firmware bertanda tangan terbaru.
  • Ponsel refurbish boot loop dengan 4013, mulai ulang tepat saat arus mencapai ~550 mA: rel daya buck/charging tidak bekerja (diagnosis bangku perbaikan nyata) — perangkat keras.
  • Driver / iTunes gagal terpasang dengan error terkait: memasang ulang driver Apple Mobile Device / AMDS lewat kit perbaikan adalah solusi paling andal.

Terkait

  • Situs resmi aisi-helper
  • Alat kode error iTunes aisi-helper
  • Panduan flash / un-brick satu klik
  • iPhone tidak terhubung ke komputer
Arah triase dirangkum dari dokumen dukungan resmi Apple (error pembaruan dan pemulihan iOS 108308 / error 9·4005·4013·4014 102561); penyebab tingkat perbaikan dirangkum dari pengalaman komunitas perbaikan publik, hanya untuk rujukan - cari perbaikan profesional untuk kerusakan perangkat keras.
