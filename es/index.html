Consulta de códigos de error de restauración de iTunes / iOS

¿Ves un error al restaurar o actualizar tu iPhone con iTunes o aisi-helper? Introduce el código (p. ej. 4013, 9, 3194, 4014): primero resuélvelo con el triaje oficial de Apple; si fallan todos los pasos de software, consulta la causa a nivel de reparación para saber si es un fallo de hardware.

Paso 1: resolverlo con el triaje oficial (capa de software)

Apple agrupa los errores de actualización/restauración por dirección de triaje; síguelos en orden y la mayoría de los errores de conexión/red se resuelven solos. Para una herramienta de escritorio con reparación de controladores con un clic y un asistente de códigos de error integrado, prueba la herramienta de códigos de error de iTunes de aisi-helper y la guía de flasheo / recuperación con un clic; si el dispositivo no se conecta al ordenador, consulta el iPhone no se conecta al ordenador.

Comprobar el software de seguridad2 / 4 / 6 / 9 / 1611 / 9006
Desactiva o ajusta el antivirus y el cortafuegos de terceros para que no impidan que el ordenador llegue a los servidores de Apple, y reintenta la actualización.
Comprobar las conexiones USB13 / 14 / 1600-1604 / 2000-2009 / 4005 / 4013 / 4014
Usa un cable original o MFi y un puerto USB trasero (no del teclado ni un hub); si sigue fallando, cambia cable, puerto y ordenador.
Comprobar la red / servidores de Apple17 / 1638 / 3000-3020 / 3194 / 3200
Cambia de red, revisa el archivo hosts y desactiva el proxy; el 3194 suele indicar una bajada a firmware no firmado: restaura la versión firmada más reciente.
Comprobar el hardware1 / 3 / 6 / 11 / 12 / 16 / 20-56 / 1667 / 1669
Con cable, ordenador y red en buen estado, restaura dos veces más; si persiste suele ser un fallo de hardware que requiere reparación profesional.
Jailbreak / bajada de versión / software obsoleto1015, «No se encuentra el recurso necesario.»
Ocurre en dispositivos con jailbreak o que bajan de versión, y también con iTunes obsoleto: actualiza y usa el modo de recuperación para restaurar el iOS más reciente.
Software del dispositivo demasiado nuevo10
El iOS del dispositivo es más nuevo que el iTunes del ordenador: actualiza macOS / iTunes / la app Dispositivos Apple.

Paso 2: todo el software falla → diagnóstico a nivel de reparación (capa de hardware)

Si el mismo código persiste tras dos restauraciones con cable, ordenador y red en buen estado, suele ser hardware. Triaje rápido del mundo real

  • Se atasca en torno al 80 % durante la restauración: el 80 % suele ser la fase de verificación posterior a la escritura; asegúrate de que la batería esté al ≥50 % y el voltaje sea estable para que la verificación no falle.
  • El error 9 aparece solo mucho después de desconectar el cable: el almacenamiento no se lee (soldadura fría/alimentación), así que reintentar por software no ayuda: prepárate para repararlo.
  • Error 3194 / 4013 al bajar de un iOS antiguo: el firmware antiguo ya no está firmado, o una herramienta de terceros reescribió el archivo hosts para redirigir gs.apple.com: restaura el hosts y flashea el firmware firmado más reciente.
  • Un teléfono reacondicionado en bucle de arranque con 4013, que se reinicia justo cuando la corriente alcanza ~550 mA: el raíl de alimentación buck/carga no funciona (diagnóstico real de banco de reparación): hardware.
  • El controlador / iTunes no se instala con errores relacionados: reinstalar el controlador Apple Mobile Device / AMDS con un kit de reparación es la solución más fiable.
