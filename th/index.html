ค้นหารหัสข้อผิดพลาดการกู้คืน iTunes / iOS

เจอข้อผิดพลาดขณะกู้คืนหรืออัปเดต iPhone ด้วย iTunes หรือ aisi-helper? ป้อนรหัส (เช่น 4013, 9, 3194, 4014) - ก่อนอื่นแก้เองตามการแยกแยะอย่างเป็นทางการของ Apple หากทุกขั้นซอฟต์แวร์ล้มเหลว ให้ดูสาเหตุระดับซ่อมเพื่อบอกว่าเป็นปัญหาฮาร์ดแวร์หรือไม่

ขั้นที่ 1: แก้เองตามการแยกแยะอย่างเป็นทางการ (ชั้นซอฟต์แวร์)

Apple จัดกลุ่มข้อผิดพลาดอัปเดต/กู้คืนตามทิศทางการแยกแยะ ทำตามลำดับแล้วข้อผิดพลาดการเชื่อมต่อ/เครือข่ายส่วนใหญ่จะหายเอง หากต้องการเครื่องมือเดสก์ท็อปที่ซ่อมไดรเวอร์คลิกเดียวและมีตัวช่วยรหัสข้อผิดพลาดในตัว ลองเครื่องมือรหัสข้อผิดพลาด iTunes ของ aisi-helper และคู่มือแฟลช / กู้เครื่องอิฐคลิกเดียว หากอุปกรณ์เชื่อมต่อคอมพิวเตอร์ไม่ได้เลย ดูiPhone เชื่อมต่อคอมพิวเตอร์ไม่ได้

ตรวจสอบซอฟต์แวร์รักษาความปลอดภัย2 / 4 / 6 / 9 / 1611 / 9006
ปิดหรือปรับแอนตี้ไวรัสและไฟร์วอลล์ของบริษัทอื่นไม่ให้บล็อกคอมพิวเตอร์เชื่อมต่อเซิร์ฟเวอร์ Apple แล้วลองอัปเดตอีกครั้ง
ตรวจสอบการเชื่อมต่อ USB13 / 14 / 1600-1604 / 2000-2009 / 4005 / 4013 / 4014
ใช้สายแท้หรือ MFi และพอร์ต USB ด้านหลัง (ไม่ใช่พอร์ตคีย์บอร์ดหรือฮับ) หากยังล้มเหลวให้เปลี่ยนสาย พอร์ต และคอมพิวเตอร์
ตรวจสอบเครือข่าย / เซิร์ฟเวอร์ Apple17 / 1638 / 3000-3020 / 3194 / 3200
เปลี่ยนเครือข่าย ตรวจไฟล์ hosts และปิดพร็อกซี 3194 มักหมายถึงการดาวน์เกรดไปเฟิร์มแวร์ที่ไม่ได้เซ็น — ให้กู้คืนบิลด์ที่เซ็นล่าสุด
ตรวจสอบฮาร์ดแวร์1 / 3 / 6 / 11 / 12 / 16 / 20-56 / 1667 / 1669
ด้วยสาย คอมพิวเตอร์ และเครือข่ายที่ดี ให้กู้คืนอีกสองครั้ง หากยังเป็นมักเป็นฮาร์ดแวร์เสียและต้องซ่อมโดยมืออาชีพ
เจลเบรค / ดาวน์เกรด / ซอฟต์แวร์ล้าสมัย1015, 'ไม่พบทรัพยากรที่จำเป็น'
เกิดกับอุปกรณ์ที่เจลเบรคหรือดาวน์เกรด รวมถึง iTunes ล้าสมัย — อัปเดตแล้วใช้โหมดการกู้คืนเพื่อกู้คืน iOS ล่าสุด
ซอฟต์แวร์อุปกรณ์ใหม่เกินไป10
iOS ของอุปกรณ์ใหม่กว่า iTunes บนคอมพิวเตอร์ — อัปเดต macOS / iTunes / แอป Apple Devices

ขั้นที่ 2: ซอฟต์แวร์ล้มเหลวทั้งหมด → วินิจฉัยระดับซ่อม (ชั้นฮาร์ดแวร์)

หากรหัสเดิมยังขึ้นหลังกู้คืนสองครั้งด้วยสาย คอมพิวเตอร์ และเครือข่ายที่ดี มักเป็นฮาร์ดแวร์ แผนภาพด้านล่างจับคู่รหัสข้อผิดพลาด → ชิ้นส่วนเมนบอร์ดจากมุมโต๊ะซ่อม (เป็นภาพประกอบเท่านั้น การวินิจฉัยจริงต้องวัดความต้านทาน/กระแส) ปล่อยให้มืออาชีพซ่อมฮาร์ดแวร์

ตารางรหัสข้อผิดพลาดที่พบบ่อย

แยกแยะกรณีจริงอย่างรวดเร็ว

  • ค้างราว 80% ระหว่างกู้คืน: 80% มักเป็นขั้นตรวจสอบหลังเขียนข้อมูล — ให้แบตเตอรี่ ≥50% และแรงดันคงที่เพื่อไม่ให้การตรวจสอบล้มเหลว
  • ข้อผิดพลาด 9 ขึ้นหลังถอดสายไปนาน: ที่เก็บข้อมูลอ่านไม่ได้ (บัดกรีหลวม/ไฟ) การลองใหม่ด้วยซอฟต์แวร์จึงไม่ช่วย — เตรียมซ่อม
  • ข้อผิดพลาด 3194 / 4013 เมื่อดาวน์เกรด iOS เก่า: เฟิร์มแวร์เก่าไม่ถูกเซ็นแล้ว หรือเครื่องมือบริษัทอื่นเขียนทับ hosts เพื่อเปลี่ยนเส้นทาง gs.apple.com — กู้คืน hosts และแฟลชเฟิร์มแวร์ที่เซ็นล่าสุด
  • เครื่องรีเฟอร์บิชวนลูปด้วย 4013 รีสตาร์ทตอนกระแสถึง ~550 mA พอดี: รางจ่ายไฟ buck/ชาร์จไม่ทำงาน (วินิจฉัยจากโต๊ะซ่อมจริง) — ฮาร์ดแวร์
  • ไดรเวอร์ / iTunes ติดตั้งไม่ได้พร้อมข้อผิดพลาดที่เกี่ยวข้อง: ติดตั้งไดรเวอร์ Apple Mobile Device / AMDS ใหม่ด้วยชุดเครื่องมือซ่อมเป็นวิธีที่เชื่อถือได้ที่สุด

ที่เกี่ยวข้อง

  • เว็บไซต์ทางการ aisi-helper
  • เครื่องมือรหัสข้อผิดพลาด iTunes ของ aisi-helper
  • คู่มือแฟลช / กู้เครื่องอิฐคลิกเดียว
  • iPhone เชื่อมต่อคอมพิวเตอร์ไม่ได้
ทิศทางการแยกแยะรวบรวมจากเอกสารสนับสนุนทางการของ Apple (ข้อผิดพลาดการอัปเดตและการกู้คืน iOS 108308 / ข้อผิดพลาด 9·4005·4013·4014 102561) สาเหตุระดับซ่อมสังเคราะห์จากประสบการณ์ชุมชนซ่อมสาธารณะ เพื่ออ้างอิงเท่านั้น - หาช่างมืออาชีพสำหรับความเสียหายของฮาร์ดแวร์
