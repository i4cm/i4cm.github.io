iTunes / iOS 刷机错误码查询

用 iTunes 或爱思助手刷机、恢复 iPhone 时报错？输入错误码（如 4013、9、 3194、4014），先按 Apple 官方排查方向自救；软件都试过仍报，再看 维修级真因判断是不是硬件问题。

第一步：按官方「排查方向」自救（软件层）

Apple 官方把更新/恢复报错按排查方向归类，照顺序逐个排，多数连接/网络类报错能自己解决。想要带一键修复驱动、内置错误码助手的桌面工具，可用 爱思助手·iTunes 错误码查询 和 一键刷机 / 救砖教程；设备干脆连不上电脑，参考 iPhone 连接不上电脑怎么办。

检查安全软件2 / 4 / 6 / 9 / 1611 / 9006
关闭或调整第三方杀毒与防火墙，别让它拦截电脑连 Apple 服务器，然后重试更新。
检查 USB 连接13 / 14 / 1600–1604 / 2000–2009 / 4005 / 4013 / 4014（及含「无效响应」）
用原装或 MFi 认证线、机箱后置 USB 口，别用键盘口/集线器；不行就换线、换口、换台电脑。
检查网络 / Apple 服务器17 / 1638 / 3000–3020 / 3194 / 3200
换网络、检查 hosts、关代理；3194 多是降级到停签固件，改刷最新在签版本。
检查硬件1 / 3 / 6 / 11 / 12 / 16 / 20–56 / 1667 / 1669 等
在线缆、电脑、网络都正常时再恢复两次；仍报多为硬件故障，需送专业维修。
越狱 / 降级 / 软件过时1015、「找不到所需资源」
越狱机或正在降级会报；iTunes 过旧也会 —— 更新后用恢复模式刷回最新版 iOS。
设备软件太新10
设备 iOS 比电脑端 iTunes 还新，更新 Mac / iTunes / Apple 设备应用即可。

第二步：软件都无效 → 维修级判因（硬件层）

如果在良好的线缆、电脑、网络下连刷两次仍报同一个码，多半是硬件问题。下面这张图是维修视角下 「错误码 → 主板部位」的常见对应（仅示意，实际需结合阻值/电流实测）。硬件维修请交专业人员。

真实场景速判

  • 刷到 80% 左右卡住报错：80% 多是固件写入后的校验阶段，先确认电量 ≥ 50%、电压稳定，避免校验失败。
  • 拔掉数据线很久后才报 9：基本是没读到硬盘（虚焊/供电），软件再试也无解，准备送修。
  • 降级旧版 iOS 报 3194 / 4013：旧固件已停签，或 hosts 被第三方工具改写重定向 gs.apple.com → 还原 hosts、改刷最新在签固件。
  • 二修机白苹果反复重启报 4013、开机电流刚过 550mA 就重启：下电源没正常工作（真实维修判断），属硬件。
  • 驱动 / iTunes 装不上连带报错：用爱思助手「工具箱 → iTunes 及驱动修复」一键重装最稳。
